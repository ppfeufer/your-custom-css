== Description ==

Add your own CSS to your WordPress Backend and Frontend.

**Features**

* Easy install
* Write your own CSS for your Frontend
* Write your own CSS for your Backend

**Translations**

* English
* German

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'Your Custom CSS'
1. Click 'install'

**Manual Installation**

1. Upload folder `your-custom-css` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin in 'Twitterbutton' menu
